Optical Music Recognition

Symphonia.io is an open source project, offering Optical Music Recognition service. It's based on Audiveris project which is responsible for the underlying OMR processing.

  • More

MusicXML Editor [experimental]

We provide one of th very few open source editor of MusicXML files. It runs directly in a web browser and is capable of importing/exporting MusicXML files, as well as, graphical editing of basic music notation.

  • More

Mobile Application

There is also a mobile application capable of interacting with Symphonia.io API. This way, you can just take a picture from your smartphone and have it OMR processed in Symphonia.
