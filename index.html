Optical Music Recognition

Symphonia.io is an open source project, offering Optical Music Recognition service. It's based on Audiveris project which is responsible for the underlying OMR processing.

Public Service [experimental]

Our public instance of Symphonia.io provides a public API for usage in other applications. Users might want to try the OMR on their own directly from our swagger API documentation.
